## 1. 设计原则

### 四大核心原则
| 原则 | 说明 |
|------|------|
| **简洁清晰** | 界面干净，不堆砌元素，让画作本身成为主角 |
| **内容优先** | 所有设计服务于内容展示，画作图片永远是视觉焦点 |
| **易于使用** | 操作直觉化，用户无需学习即可上手 |
| **文化感** | 整体气质偏向艺术、典雅，区别于普通资讯类App |

### 设计参考风格
- 留白充足，类似博物馆展厅的呼吸感
- 深色背景衬托画作（画作详情页）
- 浅色背景用于内容浏览页（首页、列表页）

## 2. 色彩规范

### 2.1 主色调

```
主色（深墨绿）：  #1A3C34   ← 导航栏、按钮、强调元素
辅色（暖金色）：  #C9A84C   ← 高亮、徽章、会员标识
背景色（米白）：  #F8F5F0   ← 页面背景，营造纸张质感
深背景（近黑）：  #141414   ← 画作详情页背景，突出画作
```

### 2.2 功能色

```
成功绿：  #4CAF50   ← 操作成功提示
警告橙：  #FF9800   ← AI生成内容标识
错误红：  #F44336   ← 错误提示
信息蓝：  #2196F3   ← 普通提示信息
```

### 2.3 文字色

```
主文字：    #1A1A1A   ← 标题、正文
次文字：    #666666   ← 辅助说明、时间、标签
禁用文字：  #BBBBBB   ← 不可点击状态
反色文字：  #FFFFFF   ← 深色背景上的文字
```

### 2.4 色彩使用示例

```
首页 / 列表页：米白背景 #F8F5F0 + 主文字 #1A1A1A
画作详情页：   深背景 #141414 + 反色文字 #FFFFFF
按钮主色：     深墨绿 #1A3C34 + 白色文字
AI生成标识：   警告橙 #FF9800 背景 + 白色文字"AI生成"
```

## 3. 字体规范

### 3.1 字体选择（全部使用系统字体，无需加载外部字体，零成本）

```
中文字体：系统默认（iOS: PingFang SC / Android: Noto Sans SC）
英文字体：系统默认（iOS: SF Pro / Android: Roboto）
```

> 💡 使用系统字体的优势：加载速度最快、零流量消耗、各平台显示效果最佳。

### 3.2 字号规范

| 用途 | 字号 | 字重 | 说明 |
|------|------|------|------|
| 页面大标题 | 22px | Bold | 博物馆名、艺术家名 |
| 画作标题 | 18px | SemiBold | 画作名称 |
| 正文内容 | 15px | Regular | 讲解文字 |
| 辅助信息 | 13px | Regular | 年代、风格、标签 |
| 小标注 | 11px | Regular | 图片来源、版权信息 |

### 3.3 行高规范

```
标题行高：字号 × 1.4
正文行高：字号 × 1.8   ← 较宽松，提升阅读舒适度
```

## 4. 间距与布局规范

### 4.1 基础间距单位

```
基础单位：8px
常用间距：8 / 16 / 24 / 32 / 48px
页面左右边距：16px
卡片内边距：12px
```

### 4.2 页面安全区域

```
顶部：状态栏高度（系统自动适配）+ 导航栏 44px
底部：TabBar 50px + 底部安全区（iPhoneX以上自动适配）
内容区左右：各 16px 边距
```

### 4.3 卡片布局规范

```
画作卡片（列表页）：
  - 宽度：屏幕宽度的 47%（两列排布）
  - 图片比例：3:4（竖向，适合大多数画作比例）
  - 圆角：8px
  - 阴影：0 2px 8px rgba(0,0,0,0.08)

艺术家卡片：
  - 宽度：屏幕宽度的 30%（三列排布）
  - 头像比例：1:1（圆形裁剪）
  - 姓名：居中，13px

博物馆卡片（横向滚动）：
  - 宽度：200px
  - 高度：120px
  - 图片：全覆盖，底部渐变遮罩 + 白色文字
  - 圆角：12px
```

## 5. 核心组件规范

### 5.1 底部导航栏 TabBar

```
背景色：#FFFFFF，顶部 1px 分割线 #EEEEEE
图标尺寸：24px
选中色：主色 #1A3C34
未选中色：#999999
文字：10px

标签页（从左至右）：
[ 首页 ]  [ 探索 ]  [ 📷 拍照 ]  [ 教育 ]  [ 我的 ]

中间"拍照"按钮为突出样式：
  - 圆形按钮，直径 56px
  - 背景色：主色 #1A3C34
  - 图标：白色相机图标
  - 位置：略高于TabBar，产生悬浮效果
```

### 5.2 画作详情页播放器

```
位置：固定在页面底部，TabBar之上
背景：半透明黑色 rgba(0,0,0,0.85)，毛玻璃效果
高度：64px
内容：[喇叭图标] [进度条] [时间] [播放/暂停按钮]
文字色：白色
进度条激活色：暖金色 #C9A84C
```

### 5.3 AI生成内容标识

```
样式：小型标签，显示在讲解内容顶部
背景：#FF9800（警告橙）
文字："⚡ AI生成内容，仅供参考"
字号：11px，白色
圆角：4px
目的：明确告知用户内容来源，建立信任
```

### 5.4 主要按钮规范

```
主按钮（如"开始聆听"）：
  - 背景：#1A3C34
  - 文字：白色，15px，SemiBold
  - 高度：44px
  - 圆角：22px（胶囊形）
  - 宽度：自适应或固定200px

次要按钮（如"收藏"）：
  - 背景：透明
  - 边框：1px solid #1A3C34
  - 文字：#1A3C34，15px
  - 高度：44px
  - 圆角：22px

文字按钮（如"查看更多"）：
  - 无背景无边框
  - 文字：#1A3C34，13px
```

### 5.5 搜索框

```
背景：#EEEEEE
圆角：20px（胶囊形）
高度：36px
内边距：左12px 右12px
左侧：搜索图标（灰色）
占位文字："搜索画作、艺术家..."，#999999
```

## 6. 页面设计规范

### 6.1 首页

```
├─────────────────────────┤

├─────────────────────────┤
│  最新收录（两列瀑布流）   │  画作卡片
```

### 6.2 画作详情页

```

┌─────────────────────────┐
│  返回按钮  画作名称  收藏  │  背景：深色 #141414
├─────────────────────────┤
│                         │
│      画作高清图片         │  可双指缩放，点击细节弹标注
│      （可交互区域）       │
│                         │
├─────────────────────────┤
│  画作名称（大字）         │  白色文字

├─────────────────────────┤
│  标签：[印象派][油画]     │
```

### 6.3 拍照识图页

```
结构：
┌─────────────────────────┐
│  返回按钮    拍照识图      │  深色背景
├─────────────────────────┤
│                         │
│      摄像头取景框         │  中间圆角矩形扫描框
│      [扫描动画线条]       │  暖金色扫描线上下移动
│                         │
├─────────────────────────┤
│  提示文字：              │
│  "将画作置于框内拍摄"     │  白色小字，居中
├─────────────────────────┤
│       [拍照按钮]          │  圆形大按钮，白色
├─────────────────────────┤
│  或 [从相册选择]          │  文字按钮
└─────────────────────────┘

识别结果：
- 识别成功 → 直接跳转画作详情页，顶部短暂提示"已识别：蒙娜丽莎"
- 识别失败 → 提示"未能识别，请确保画作清晰可见"+ 显示手动搜索框
- 数据库无结果 → 自动调用AI查询，显示AI生成内容（带橙色标识）
```

## 8. 动效规范

### 原则：轻量、流畅、不干扰内容

| 场景 | 动效 | 时长 |
|------|------|------|
| 页面切换 | 右侧滑入 | 300ms |
| 卡片点击 | 轻微缩放（scale 0.97） | 100ms |
| 图片加载 | 渐显（opacity 0→1） | 200ms |
| 拍照扫描线 | 上下循环移动 | 1500ms/次 |
| 语音进度条 | 平滑过渡 | 实时 |
| 收藏成功 | 心形图标放大弹跳 | 400ms |
| TabBar切换 | 图标轻微上移+颜色变化 | 150ms |
